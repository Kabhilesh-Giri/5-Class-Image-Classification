# Neural Network Image Classification – Project

## Overview
This project explores neural network architectures for image classification on a 5-class flower dataset.

## Experiments
- EXP1: Linear Neural Network
- EXP2: Non-linear Neural Network
- EXP3: Optimized Neural Network (Hyperparameter Tuning, BatchNorm, Dropout, Early Stopping)

## How to Run
1. Open the notebooks in the `experiments/` folder
2. Run cells sequentially
3. Ensure dataset path is correctly set
